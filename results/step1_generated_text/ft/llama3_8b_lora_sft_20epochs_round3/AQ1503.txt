Question:
What is the best performing model benchmarking the Supervised: dataset in terms of SemEval 2013 metric?

Generated SPARQL:
 SELECT DISTINCT?model?model_lbl
    WHERE {
     ?metric     a       orkgc:Metric;
                  rdfs:label ?metric_lbl.
      FILTER (str(?metric_lbl) = "SemEval 2013")
     ?benchmark      orkgp:HAS_DATASET      ?dataset.
      FILTER (str(?dataset) = "Supervised:")
     ?cont         orkgp:HAS_BENCHMARK     ?benchmark.
     ?cont         orkgp:HAS_EVALUATION    ?eval.
     ?eval         orkgp:HAS_VALUE         ?value;
                    orkgp:HAS_METRIC        ?metric.
     ?cont         orkgp:HAS_MODEL         ?model.
     ?model      rdfs:label              ?model_lbl.
     ?cont      orkgp:HAS_EVALUATION    ?eval2.
     ?eval2     orkgp:HAS_VALUE         ?value2;
                 orkgp:HAS_METRIC        ?metric2.
      FILTER (
       ?metric =?metric2
        &&?value >=?value2
      )
    }
    ORDER BY DESC(?value)
    LIMIT 1<|eot_id|>
