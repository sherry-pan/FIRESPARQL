Question:
What is the top benchmark result (metric and value) over the dataset ACE 2004?

Generated SPARQL:
 SELECT DISTINCT?metric?metric_lbl (MAX(?value) AS?score)
     WHERE {
      ?dataset       a                orkgc:Dataset;
                      rdfs:label      ?dataset_lbl.
           FILTER (str(?dataset_lbl) = "ACE 2004")
        ?benchmark      orkgp:HAS_DATASET      ?dataset;
                        orkgp:HAS_EVALUATION   ?eval.
        ?paper        orkgp:HAS_BENCHMARK     ?benchmark.
         OPTIONAL {?paper      orkgp:HAS_METRIC        ?metric.
                 ?metric      rdfs:label              ?metric_lbl.}
        ?eval         orkgp:HAS_PAPER         ?paper.
         BIND(xsd:int(?eval(?eval:VALUE     ?value).) AS?value)
     }
     ORDER BY DESC(?score)
     LIMIT 1<|eot_id|>
