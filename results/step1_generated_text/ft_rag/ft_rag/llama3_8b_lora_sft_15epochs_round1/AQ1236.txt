Can you provide the highest benchmark result, including the metric and score, for the Gibson PointGoal Navigation dataset?

Generated SPARQL:

     WHERE {
      ?dataset       a                orkgc:Dataset;

      ?benchmark      orkgp:HAS_DATASET      ?dataset;
                       orkgp:HAS_EVALUATION   ?eval.
       OPTIONAL {?eval           orkgp:HAS_METRIC        ?metric.
               ?metric          rdfs:label              ?metric_lbl.}
      ?cont         orkgp:HAS_BENCHMARK     ?benchmark;
                     orkgp:HAS_EVALUATION     ?eval.
           OPTIONAL {?cont      orkgp:HAS_METHOD     ?method.
                    ?method      rdfs:label          ?method_lbl.}
           OPTIONAL {?cont      orkgp:HAS_EVALUATION     ?eval2.
                    ?eval2      orkgp:HAS_VALUE        ?value.}
           OPTIONAL {?cont      orkgp:HAS_VALUE          ?value.
                    ?value      orkgp:P32               ?value_lbl.}
     }
     ORDER BY DESC(?score)<|eot_id|>
