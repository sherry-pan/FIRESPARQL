Question:
What is the most common location in the studies?

Generated SPARQL:
 SELECT DISTINCT?location?location_lbl
     WHERE {
      ?dataset       a                orkgc:Dataset;
                      rdfs:label      ?dataset_lbl.
       FILTER (str(?dataset_lbl) = "Most common location in the studies")
      ?dataset       orkgp:HAS_DATASET      ?study.
      ?study         orkgp:HAS_LOCATION      ?location.
      ?location      rdfs:label              ?location_lbl.
     }     ORDER BY DESC(?location_lbl) LIMIT 1.
