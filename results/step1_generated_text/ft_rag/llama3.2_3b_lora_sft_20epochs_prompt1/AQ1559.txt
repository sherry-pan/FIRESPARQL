Question:
Which model has achieved the highest Accuracy score on the SST-5 Fine-grained classification benchmark dataset?

Generated SPARQL:
 SELECT?model?model_lbl
     WHERE {
      ?dataset       a                orkgc:Dataset;
                      rdfs:label      ?dataset_lbl.
        FILTER (str(?dataset_lbl        = "SST-5 Fine-grained classification")
                )
       ?benchmark      orkgp:HAS_DATASET      ?dataset;
                      orkgp:HAS_EVALUATION     ?eval.
       ?eval           orkgp:P71081           ?accuracy.
        OPTIONAL {?eval           orkgp:P32           ?method.}
       ?cont         orkgp:HAS_BENCHMARK     ?benchmark.
        OPTIONAL {?cont      orkgp:HAS_METHOD        ?method.
                  orkgp:P32               ?method.}
        OPTIONAL {?cont          orkgp:P31           ?model.
                  rdfs:label              ?model_lbl.}
        ORDER BY DESC(?accuracy)
        LIMIT 1
     }<|eot_id|>
