Question:
Where can I find code references in papers that have used the MMV model for benchmarking purposes?

Generated SPARQL:
 PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>
     PREFIX orkg: <http://orkg.org/>
     PREFIX paper: <http://orkg.org/paper/>
     PREFIX code: <http://orkg.org/code/>
     PREFIX mmv: <http://orkg.org/model/MMV>
     SELECT?paper?code
     WHERE {
      ?paper a paper:Paper.
      ?paper orkg:usesModel mmv:MMV.
      ?paper orkg:hasCode?code.
      ?code a code:CodeReference.
     }<|eot_id|>
