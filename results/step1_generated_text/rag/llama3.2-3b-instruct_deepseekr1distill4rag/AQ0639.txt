Question:
List the title and ID of research papers that contain a benchmark over the Ohsumed dataset?

Generated SPARQL:
 ```sparql
SELECT?paper?paperId 
WHERE {
 ?paper a ora:ResearchPaper.
 ?paper ora:containsBenchmark?benchmark.
 ?benchmark ora:usesDataset?dataset.
 ?dataset rdfs:label?datasetLabel.
  FILTER (str(?datasetLabel) = "Ohsumed dataset"^^xsd:string).
  SERVICE orkgr:dataset {
   ?benchmark ora:usesDataset?dataset.
   ?dataset rdfs:label?datasetLabel.
  }
}
```
