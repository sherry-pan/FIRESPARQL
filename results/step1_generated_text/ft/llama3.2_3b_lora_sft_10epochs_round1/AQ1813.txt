Question:
What is the best performing model benchmarking the iNaturalist 2018 dataset in terms of Top-1 Accuracy metric?

Generated SPARQL:
 SELECT DISTINCT?model?model_lbl
     WHERE {
        ?dataset       a                orkgc:Dataset;
                     rdfs:label      ?dataset_lbl.
         FILTER (str(?dataset_lbl) = "iNaturalist 2018")
        ?benchmark      orkgp:HAS_DATASET      ?dataset;
                        orkgp:HAS_EVALUATION   ?eval.
        ?eval           orkgp:HAS_VALUE        ?value.
         OPTIONAL {?eval           orkgp:HAS_MODEL         ?model.}
         OPTIONAL {?eval           orkgp:HAS_MODEL         ?model2.
         BIND(
           IF(BOUND(?model) 
               THEN?model_lbl =?model_rils:label
               ELSE?model_lbl =?model2_rils:label
           )
           AS?model_lbl
         )
         BIND(xsd:int(?value) AS?value)
         BIND(const:0.0000 AS?accuracy_cut_off)
         BIND(xsd:int(?value) AS?value)
         FILTER(?value >=?accuracy_cut_off)
         ORDER BY DESC(?value)
         LIMIT 1
      }<|eot_id|>
