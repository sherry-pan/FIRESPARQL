Question:
List the metrics that are used to evaluate models on the Yelp-5 benchmark dataset?

Generated SPARQL:
 PREFIX orkg: <http://orkg.org/ontology/>
     SELECT?metric
     WHERE {
      ?metric orkg:evaluationMetricFor {
         rdfs:label?label.
         orkg:dataset "Yelp-5".
         orkg:benchmark "5".
         filter (str(?label) = "Accuracy" ) || 
                (str(?label) = "Precision" ) || 
                (str(?label) = "Recall" ) || 
                (str(?label) = "F1 score" ) || 
                (str(?label) = "AUC-ROC" )
       }
     } 
     LIMIT 5 
     ORDER BY DESC 
     FETCH ALL
