Question:
What is the best performing model benchmarking the Oxford-IIIT Pets dataset in terms of FLOPS metric?

Generated SPARQL:
 SELECT DISTINCT?model?model_lbl
     WHERE {
        ?dataset       a                orkgc:Dataset;
                     rdfs:label      ?dataset_lbl.
         FILTER (str(?dataset_lbl) = "Oxford-IIIT Pets")
        ?benchmark      orkgp:HAS_DATASET      ?dataset;
                        orkgp:HAS_EVALUATION   ?eval.
        ?paper         orkgp:HAS_BENCHMARK     ?benchmark.
        ?model        orkgp:HAS_PAPER         ?paper;
                       orkgp:HAS_MODEL         ?model.
        ?model      rdfs:label              ?model_lbl.
        ?model      orkgp:HAS_METRIC        ?metric.
         BIND(xsd:int(?metric) AS?metric_lbl)
         FILTER (str(?metric_lbl) = "FLOPS")
     }
     ORDER BY DESC(?metric_lbl)
     LIMIT 10<|eot_id|>
